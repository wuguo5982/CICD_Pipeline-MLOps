# Recommended Public Data Sources

Use these sources to replace or augment the synthetic demo data.

1. CMS Medicare Claims Synthetic Public Use Files (SynPUF)
   - Good for learning Medicare claim structures and safe experimentation.

2. Synthetic Medicare Enrollment, Fee-for-Service Claims, and Prescription Drug Event datasets from data.cms.gov
   - Useful for realistic synthetic enrollment and claims records.

3. data.medicaid.gov
   - Medicaid and CHIP public datasets.

4. CMS Program Integrity and FWA Guidance PDFs
   - Use as RAG knowledge-base documents.

5. Synthea
   - Synthetic EHR records, encounters, medications, and optional clinical notes.

6. Kaggle CMS Synthetic Medicare Claims Datasets
   - Useful for supervised ML experiments and labels when available.
